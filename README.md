# WordGuess

Live chat games for TikTok &amp; Twitch streamers. Word Guess, Scramble, Tug of War and Drop 4 — interactive OBS overlays your viewers play in real time, straight from chat.

**▶ Website:** _enable GitHub Pages to get your link_
**⬇ Download (Windows):** see the [Releases](../../releases) page

## What it is
A local Windows app that runs a game overlay for OBS (or any browser-source-capable software). Connect TikTok and/or Twitch, pick a mode, and your chat plays live on stream.

- **Free:** every game mode, 1 hour/day of live-connected play, one platform at a time.
- **Pro:** unlimited play, TikTok + Twitch at once, and a shareable overlay link.

## Install
1. Download the latest `WordGuess-Windows.zip` from [Releases](../../releases).
2. Unzip it and double-click `WordGuess.exe`.
3. Windows SmartScreen may warn (the app isn't code-signed yet) — click **More info → Run anyway**.
4. Your control panel opens automatically. Add `http://localhost:3000/overlay` as a browser source in OBS.
